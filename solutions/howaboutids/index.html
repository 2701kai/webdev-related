Classy and identified

ID's should be used, when you want to target a very specific element in your html file. Whereas Classes should be preferred when you intend to alter the styling of several elements, i.e. "change the color of all paragraphs" F.Y.I.: ID vs. Class

ID:
Syntax: #idname { style properties }
Names work. See CSS ID

CLASS:
Syntax: .class { background-color: yellow; }
See CSS .class selector
